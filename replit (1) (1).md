# بوت تليجرام - مدير البوتات المتعددة

## نظرة عامة
بوت تليجرام يسمح للمستخدمين برفع وتشغيل عدة بوتات، كل بوت منفصل بملفاته ومكتباته الخاصة.

## الميزات
- `/start` - بدء استخدام البوت
- إنشاء عدة بوتات منفصلة لكل مستخدم
- كل بوت له مجلد خاص بملفاته ومكتباته
- تشغيل/إيقاف البوتات
- عرض السجلات
- تثبيت مكتبات (من قائمة أو يدوياً)
- إدارة الملفات (عرض/حذف)
- رفع ملفات ZIP أو .py

## هيكل الملفات
```
user_bots/
  {user_id}/
    {bot_name_1}/
      app.py
      log.txt
      (مكتبات منفصلة)
    {bot_name_2}/
      app.py
      log.txt
      (مكتبات منفصلة)
```

## المتطلبات
- python-telegram-bot
- python-dotenv

## الملفات
- `main.py` - الكود الرئيسي للبوت
- `processes.json` - قاعدة بيانات لتتبع العمليات الشغالة
